ระบบตลาดซื้อขายผลไม้ เป็นระบบที่เริ่มพัฒนาตั้งแต่วันที่ 14 พฤษจิกายน 2565 จนถึงวันที่ 17 กุมภาพันธ์ 2566
เป็นโปจเจคจบการศึกษา ระดับชั้นปีที่ 4 ระบบซื้อขายสามารถทำงานดังนี้
